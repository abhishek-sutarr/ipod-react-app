# iPod React Application

A modern recreation of Apple's classic iPod interface built using React.js.

## Features

- Circular wheel navigation
- Interactive menu system
- Music screen
- Settings screen
- Games screen
- Cover Flow screen
- Smooth navigation experience

## Tech Stack

- React.js
- JavaScript
- HTML
- CSS

## Installation

```bash
npm install
npm start
```

## Project Structure

src/
public/
package.json
